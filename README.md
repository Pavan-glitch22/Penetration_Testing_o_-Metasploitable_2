#  Penetration Testing on Metasploitable 2

##  Overview
This project demonstrates a complete penetration testing process performed on Metasploitable 2 using Kali Linux and Metasploit Framework. The assessment covers all major phases including reconnaissance, scanning, enumeration, vulnerability assessment, and exploitation in a controlled lab environment.

---

##  Objective
- Perform reconnaissance and network scanning  
- Identify open ports and running services  
- Enumerate system and service information  
- Analyze vulnerabilities using scanning tools  
- Exploit vulnerable services using Metasploit  
- Understand real-world penetration testing workflow  

---

##  Tools Used
- Kali Linux  
- Nmap  
- Nessus Essentials  
- Metasploit Framework  

---

##  Key Activities
- Host discovery and service detection using Nmap  
- SMB and web enumeration  
- Vulnerability assessment using Nessus (68 vulnerabilities found)  
- Exploitation using Metasploit modules  

---

##  Key Findings
- FTP backdoor vulnerability (vsftpd 2.3.4)  
- UnrealIRCd backdoor vulnerability  
- Samba misconfiguration and null session access  
- Weak password policies  
- Insecure services (Telnet, VNC)  
- Multiple critical vulnerabilities identified  

---

##  Exploitation
- Gained remote access using Metasploit  
- Established Meterpreter sessions  
- Executed commands on target system  
- Accessed system files and information
